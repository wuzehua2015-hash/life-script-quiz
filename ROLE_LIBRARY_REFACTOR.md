# 人生剧本测试v2.0 - 角色库重构方案

## 一、核心问题诊断

### 1. 原型Cast描述问题
当前cast描述是"通用模板"，不符合具体角色：
- 孤勇者的"高期待施压者"不适合甄嬛（甄远道是温和宠女父亲）
- 需要为每个原型设计更准确的"先天配角"描述

### 2. 角色原型归属问题
- 林黛玉放在"讨好者"不太准确——她更多是"敏感+疏离"，而非讨好
- 祥子从"奋斗者"到"受害者"的转变需要更精确的原型归属
- 部分角色跨原型（如甄嬛既是孤勇者也是控制狂），需要明确主次

### 3. 重复角色问题
- 悉达多：同时在隐士和觉醒者中
- 贾宝玉：同时在隐士和觉醒者中
- 需要删除重复，明确每个角色的唯一归属

## 二、12原型重新定义与Cast描述

### 1. 孤勇者 (Lone Hero)
**核心特征**: 独自战斗、背负使命、隐忍不发、为信念燃烧自己
**四维**: achievement + battle + perfection + chasing

**先天配角（父母/原生家庭）**:
- 缺席或无力保护者：想帮你但无能为力（如甄嬛父母——在宫外无法保护她）
- 过早托付重任者：把家族/使命责任压在你肩上（如梅长苏的师父、少安的奶奶）
- 高期待但低支持：要求你成功却不给资源（如Tony Stark的父亲）

**后天配角**:
- 想靠近却被推开的同行者
- 仰望但无法亲近的追随者
- 必须独自面对的强大敌人

### 2. 讨好者 (Pleaser)
**核心特征**: 用付出来换取爱、害怕冲突、自我价值感低、委屈求全
**四维**: relationship + victim + inferiority + fate

**先天配角**:
- 情感忽视者：你的感受从未被看见（如松子父亲）
- 条件式关爱者：只有乖/听话才配得到爱（如樊胜美母亲）
- 索取者：把你当情绪垃圾桶或经济来源

**后天配角**:
- 享受你的好从不回报的利用者
- 想帮你但你拒绝被帮的拯救者

### 3. 隐士 (Hermit)
**核心特征**: 与世界保持距离、独处充电、害怕受伤、观察者心态
**四维**: security + detachment + inferiority/lost + stagnation

**先天配角**:
- 过度保护者：世界很危险，待在安全的地方
- 情感疏离者：亲近意味着受伤，保持距离
- 不理解你的期待者：为什么你不像别人一样外向

**后天配角**:
- 物理上很近心理上很远的室友/同事
- 偶尔联系的网友（安全距离的关系）

### 4. 控制狂 (Controller)
**核心特征**: 必须掌控一切、厌恶失控、计划性强、不信任他人
**四维**: achievement + control + perfection + chasing

**先天配角**:
- 混乱无序的家庭环境：你发誓不要像他们那样失控
- 突然失去控制的经历：让你发誓要掌控自己的人生

**后天配角**:
- 被你控制的傀儡（后来变成反抗者）
- 阳奉阴违的执行者

### 5. 受害者 (Victim)
**核心特征**: 觉得世界不公平、无力感、被动接受、等待拯救
**四维**: relationship + victim + inferiority + fate

**先天配角**:
- 加害者：你的剧本从受伤开始
- 无力保护者：看着你受伤却无能为力

**后天配角**:
- 被你受害者气场吸引来的施虐者
- 想帮你但你拒绝改变的拯救者

### 6. 表演者 (Performer)
**核心特征**: 需要被关注、害怕平庸、形象管理、内心空虚
**四维**: unique + narcissism + perfection + exploration

**先天配角**:
- 只有表现好才配被爱的父母
- 忽视你的感受只赞美成就的人

**后天配角**:
- 后来厌倦离开的观众
- 和你争夺关注的人

### 7. 拯救者 (Savior)
**核心特征**: 通过帮助他人获得价值、被需要才有存在感、忽视自己
**四维**: service + cooperation + inferiority + creation

**先天配角**:
- 需要被照顾的弱者（你从小学会了照顾）
- 牺牲型榜样（妈妈/爸爸就是这样付出的）

**后天配角**:
- 依赖你的被拯救者
- 甩锅者（知道你会接手）

### 8. 漫游者 (Wanderer)
**核心特征**: 拒绝束缚、不愿安定、寻找自我、害怕深入
**四维**: unique + detachment + lost + exploration

**先天配角**:
- 缺席者：你学会了不依赖任何人
- 无法理解的旁观者：他们希望你稳定

**后天配角**:
- 短暂的同行者（来了又走）
- 散落各地的旧识

### 9. 战士 (Warrior)
**核心特征**: 战斗本能、荣誉至上、保护弱者、愿意牺牲
**四维**: achievement + battle + authenticity + creation

**先天配角**:
- 坚强的后盾：支持你去战斗
- 共同战斗的战友：教会你战斗的方式

**后天配角**:
- 可以托付后背的搭档
- 尊重的敌人

### 10. 治愈者 (Healer)
**核心特征**: 治愈他人创伤、温暖包容、倾听支持、给予力量
**四维**: relationship + cooperation + authenticity + creation

**先天配角**:
- 真实的关爱者：给你无条件的爱
- 情感支持的榜样：教会你如何爱人

**后天配角**:
- 平等的同行者（互相滋养）
- 受益于你帮助的人

### 11. 观察者 (Observer)
**核心特征**: 深度观察、理性分析、保持距离、知识丰富但体验少
**四维**: security + detachment + authenticity + exploration

**先天配角**:
- 尊重边界的支持者：给你独处的空间
- 智慧的引导者：教会你观察和思考

**后天配角**:
- 受益于你智慧的知己
- 独立但亲密的伴侣

### 12. 觉醒者 (Awakened)
**核心特征**: 看透剧本运作、自我意识强、在觉醒和融入间摇摆
**四维**: authenticity + cooperation + authenticity + creation

**先天配角**:
- 觉醒触发者：他们的痛苦或智慧启发了你
- 被重新理解的真实的人：你看到了他们的局限和尽力

**后天配角**:
- 共同成长的伙伴
- 互相启发的觉醒者

## 三、角色重新分配方案

### 需要调整的角色：

1. **林黛玉** → 从讨好者移到**隐士**
   - 理由：黛玉不是讨好型，她是敏感+疏离+沉浸在自己的世界
   - 四维：unique + detachment + lost + fate

2. **甄嬛** → 只在**孤勇者**中保留，从控制狂中删除
   - 理由：甄嬛的核心是"独自战斗、背负使命"，控制是生存手段而非本质

3. **祥子** → 前期在**战士**，后期在**受害者**
   - 需要拆分或选择最能代表他的阶段

4. **删除重复**:
   - 隐士中的悉达多 → 删除（保留觉醒者中的悟道后悉达多）
   - 隐士中的贾宝玉 → 删除（保留觉醒者中的悟道后宝玉）
   - 控制狂中的甄嬛（后期）→ 删除

5. **新增角色补充**:
   - 讨好者需要更多男性角色（当前只有祥子、许三观）
   - 战士需要更多女性角色
