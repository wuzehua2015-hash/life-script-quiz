# 人生剧本导航设计分析

## 参考来源
哈利波特官网 (harrypotter.com) 导航设计分析

---

## 1. 整体结构

```
┌─────────────────────────────────────────────────────────────┐
│  [社交图标]          [Logo]          [搜索] [登录] [注册]     │  ← 顶部栏 (70px)
├─────────────────────────────────────────────────────────────┤
│  首页 ▼ | 新闻 ▼ | 档案 | 发现 ▼ | 分类 | 画像 | 体验 | 商店  │  ← 主导航 (50px)
└─────────────────────────────────────────────────────────────┘
```

---

## 2. 顶部栏设计 (Header Bar)

### 尺寸与布局
- **高度**: 70px
- **背景色**: `#0a0a1a` (深邃黑蓝)
- **底部边框**: 1px solid `rgba(212, 175, 55, 0.2)` (淡金色)
- **布局**: Flexbox, justify-content: space-between
- **内边距**: 0 40px (PC), 0 20px (Mobile)

### 左侧 - 社交图标
- **图标**: YouTube, TikTok, Facebook, Instagram
- **尺寸**: 20px × 20px
- **颜色**: `#888` → 悬停 `#d4af37` (金色)
- **间距**: 16px

### 中间 - Logo
- **位置**: 绝对居中 (absolute center)
- **高度**: 40px
- **内容**: "人生剧本" 艺术字

### 右侧 - 功能按钮
- **搜索框**: 
  - 宽度: 200px
  - 背景: `rgba(255,255,255,0.1)`
  - 圆角: 20px
  - 占位符: "Search..."
  
- **登录/注册按钮**:
  - 登录: 透明背景 + 白色边框
  - 注册: 金色背景 `#d4af37` + 深色文字
  - 圆角: 4px
  - 内边距: 8px 20px

---

## 3. 主导航设计 (Main Navigation)

### 尺寸与布局
- **高度**: 50px
- **背景色**: `#0f0f2e` (深蓝)
- **布局**: Flexbox, justify-content: center
- **内边距**: 0

### 导航项样式
- **字体**: 14px, weight 500, 大写
- **字间距**: 1.5px
- **颜色**: `#ffffff`
- **悬停颜色**: `#d4af37` (金色)
- **内边距**: 0 20px
- **过渡**: color 0.3s ease

### 下拉菜单指示器
- **图标**: ▼ (向下箭头)
- **尺寸**: 8px
- **颜色**: 继承父级
- **左边距**: 6px

### 下拉菜单
- **背景**: `#1a1a3e` (半透明深蓝)
- **边框**: 1px solid `rgba(212, 175, 55, 0.3)`
- **阴影**: 0 10px 40px rgba(0,0,0,0.5)
- **圆角**: 0 0 8px 8px
- **最小宽度**: 200px
- **动画**: 淡入 + 向下滑动 0.2s

### 下拉菜单项
- **内边距**: 12px 20px
- **字体**: 13px, weight 400
- **颜色**: `#ccc`
- **悬停**: 背景 `rgba(212, 175, 55, 0.1)`, 颜色 `#d4af37`

## 4. 移动端设计

### 汉堡按钮
- **位置**: 左上角
- **尺寸**: 24px × 24px
- **颜色**: `#ffffff`
- **三条线样式**:
  - 宽度: 24px
  - 高度: 2px
  - 间距: 6px
  - 背景: 白色

### 抽屉菜单
- **宽度**: 280px
- **背景**: `#0f0f2e`
- **位置**: 左侧滑入
- **动画**: transform translateX(-100%) → translateX(0), 0.3s ease
- **遮罩**: rgba(0,0,0,0.7)

### 抽屉菜单结构

## 6. 字体规范

| 元素 | 大小 | 字重 | 样式 |
|------|------|------|------|
| Logo | 24px | 700 | 艺术字 |
| 导航项 | 14px | 500 | 大写, 字间距1.5px |
| 下拉项 | 13px | 400 | 正常 |
| 移动端菜单 | 16px | 400 | 正常 |
| 按钮 | 14px | 600 | 大写 |

---

## 7. 动画效果

### 悬停效果
- **颜色过渡**: 0.3s ease
- **背景过渡**: 0.2s ease

### 下拉菜单
- **显示**: opacity 0→1, translateY(-10px)→0, 0.2s ease
- **隐藏**: opacity 1→0, translateY(0)→(-10px), 0.15s ease

### 移动端抽屉
- **打开**: translateX(-100%)→0, 0.3s ease
- **关闭**: translateX(0)→(-100%), 0.25s ease

### 汉堡按钮变形
- **打开**: 中间线淡出, 上下线旋转45度形成X
- **动画**: 0.3s ease

---

## 8. 响应式断点

| 断点 | 宽度 | 行为 |
|------|------|------|
| Desktop | > 1024px | 完整横向导航 |
| Tablet | 768px - 1024px | 简化导航, 部分项收起 |
| Mobile | < 768px | 汉堡菜单 + 抽屉 |
