# 人生剧本测试

基于PTK「人生剧本理论」的H5互动测试网页。

## 🎬 理论框架

每个人的人生都是一部自编自导自演的电影：
- **编剧**：潜意识创作剧本
- **导演**：执行控场
- **演员**：角色演绎

### 四个维度
1. **核心驱动力** - 为什么而活
2. **与世界的关系** - 如何看待外界
3. **与自我的关系** - 如何看待自己
4. **与时间的关系** - 如何看待人生进程

### 12种人设原型
孤勇者、讨好者、隐士、控制狂、受害者、表演者、拯救者、漫游者、战士、治愈者、观察者、觉醒者

## 🎮 测试特色

- **剧本杀风格**：12个生活化场景，有角色对话和选择
- **电影海报结果页**：展示你的专属人生剧本
- **四维雷达图**：可视化你的心理画像
- **温柔吐槽**：发现旧模式的荒谬之处
- **行动指南**：明日即可执行的改写计划

## 📝 技术栈

- 纯HTML/CSS/JavaScript，无框架依赖
- 响应式设计，适配移动端
- Canvas绘制雷达图
- 电影胶片风格UI

## 🎨 设计理念

- 禁止生硬心理学术语
- 保持剧本格式（短句、分镜感）
- 结果要让用户有「被看穿了」的惊叹感
- 提供「选角内幕」揭秘感
