## 核心理论

### 人生剧本理论
每个人的人生都是一部「自编自导自演的人设使命剧」：
- **编剧**: 潜意识的创作者，设定"你是谁、你会经历什么"
- **导演**: 执行与控场，筛选配角、引导剧情
- **演员**: 角色演绎，把剧本演成日常生活

### 四个维度
1. **核心驱动力** - 为什么而活（成就/关系/安全/独特/服务）
2. **与世界的关系** - 如何看待外界（战斗/受害者/合作/疏离/掌控）
3. **与自我的关系** - 如何看待自己（完美/自卑/自恋/真实/迷失）
4. **与时间的关系** - 如何看待人生（追赶/停滞/探索/宿命/创造）

### 12种人设原型
孤勇者、讨好者、隐士、控制狂、受害者、表演者、拯救者、漫游者、战士、治愈者、观察者、觉醒者

## 技术实现

### 前端技术栈
- 纯 HTML/CSS/JavaScript，无框架依赖
- 响应式设计，适配移动端
- Canvas 绘制雷达图
- 电影胶片风格 UI
- html2canvas 生成分享海报

### 核心算法（已优化）
1. **计分系统**: 4维度 × 3题，每题5选项对应5种类型
2. **原型匹配**: 计算与12个原型的匹配度百分比
3. **动态描述**: 根据4维组合智能生成个性化描述
4. **混合原型**: 支持接近原型组合描述

### 新增功能
- **返回上一题**: 支持修改答案
- **匹配度显示**: 显示与最佳匹配原型的百分比
- **动态画像**: 根据4维组合生成专属描述
- **真实分享**: 生成可长按保存的PNG图片
- **理论介绍**: 开场增加测试说明和价值介绍

### 结果呈现
- 🎭 你的专属画像（动态生成）
- 📊 原型匹配度（百分比）
- 🎬 当前主演的烂片（温柔吐槽旧模式）
- 💭 日常场景代入（4个具体生活场景）
- 🔍 四维详细解读（每个维度都有击中感描述）
- 👨‍👩‍👧 先天配角（父母/原生家庭）
- 👥 后天配角（朋友/亲密关系/同事/上司）
- 📝 新剧本大纲（改写方向）
- 🎬 明日拍摄计划（具体行动建议）
- 📊 四维雷达图（可视化心理画像）

## 常见问题

### 1. 中文引号导致语法错误
**问题**: data.js 中的中文引号 "..." 嵌套在字符串中导致 JS 语法错误

**解决**: 将中文引号替换为书名号「...」或使用转义

### 2. GitHub Pages 缓存
**问题**: 更新后网页未立即生效

**解决**: 等待 2-5 分钟，或强制刷新 (Ctrl+F5)

### 3. 移动端适配
**问题**: 小屏幕显示不正常

**解决**: 使用 viewport meta 标签 + 响应式 CSS
