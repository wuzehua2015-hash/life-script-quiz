# 人生剧本测试系统 v2.0 - 升级文档

## 版本信息
- **版本号**: v2.0
- **升级日期**: 2026-02-24
- **升级类型**: 重大升级

## 升级内容

### 1. 角色系统重构
- **保留**: 12个核心原型
- **新增**: 80个影视/文学角色库
- **混合原型**: 支持双原型混合匹配

### 2. 前置问题
- 性别选择（男/女/多元）
- 年龄段（18-25/26-35/36-45/46+）
- 职业类型（学生/职场新人/中层管理/高管/创业者/自由职业）
- 人生阶段（求学/职场打拼/事业稳定/转型期/退休）

### 3. 算法优化
```
匹配度 = 原型匹配(40%) + 性别匹配(15%) + 年龄匹配(15%) + 职业匹配(15%) + 人生阶段匹配(15%)
```

### 4. 报告升级
- 角色匹配卡片（剧照+台词）
- 深度分析报告
- 社交分享海报
