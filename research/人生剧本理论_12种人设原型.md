# 人生剧本理论 - 12种主角人设原型

> 理论原创：PTK（夜航船纪事）
> 整理时间：2026-02-24

---

## 核心理论框架

```
主角人设（核心设定）
    ↓
编剧自动写剧情 + 设计角色（不是具体的人）
    ↓
导演按角色需求筛选演员（真实的人）
    ↓
演员拿到角色剧本开始演
```

**关键洞察**：
- 我们抱怨的是「角色」，不是「演员」
- 改写剧本从「编剧」入手——改写主角人设
- 配角分两类：
  - **先天配角**（父母）：演员固定，只能改角色
  - **后天配角**（伴侣/朋友）：演员可选，也可改角色

---

## 四个维度

### 维度一：核心驱动力（为什么而活）
- **成就型**：「我必须成功才有价值」
- **关系型**：「我必须被爱才有价值」
- **安全型**：「我必须稳定才有价值」
- **独特型**：「我必须特别才有价值」
- **服务型**：「我必须有用才有价值」

### 维度二：与世界的关系（如何看待外界）
- **战斗型**：「世界是战场，我必须赢」
- **受害者型**：「世界伤害我，我很无助」
- **合作型**：「世界是伙伴，可以共赢」
- **疏离型**：「世界与我无关，我观察」
- **掌控型**：「世界可预测，我能控制」

### 维度三：与自我的关系（如何看待自己）
- **完美型**：「我必须没有缺点」
- **自卑型**：「我本质上不够好」
- **自恋型**：「我本质上优于他人」
- **真实型**：「我有优缺点，整体OK」
- **迷失型**：「我不知道我是谁」

### 维度四：与时间的关系（如何看待人生进程）
- **追赶型**：「我必须赶上/超越」
- **停滞型**：「我已经定型，无法改变」
- **探索型**：「人生是旅程，可以体验」
- **宿命型**：「一切注定，我无能为力」
- **创造型**：「未来由我书写，可以创造」

## 12种主角人设原型

---

### 1. 孤勇者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 成就型 + 战斗型 + 完美型 + 追赶型 |
| **人设描述** | 「我必须成功，世界是战场，我不能有缺点，必须赶上」 |
| **核心信念** | 只有赢才能证明价值，休息是软弱 |
| **编剧写的剧本** | 筚路蓝缕、困难副本、孤军奋战、永不满足 |
| **常见剧情** | 创业九死一生、职场拼命三郎、感情因工作破裂、身体透支 |
| **先天配角角色** | 父母 = 「不理解但无奈的支持者」或 「高期待的施压者」 |
| **后天配角角色** | 伴侣 = 「被忽视的同行者」；朋友 = 「仰望但无法亲近的追随者」；对手 = 「必须击败的敌人」 |

---

### 2. 讨好者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 关系型 + 受害者型 + 自卑型 + 宿命型 |
| **人设描述** | 「我必须被爱，世界伤害我，我不够好，一切注定」 |
| **核心信念** | 只有付出才能换来爱，受伤是常态 |
| **编剧写的剧本** | 真心付出被辜负、反复遇到索取者、自我牺牲换认可 |
| **常见剧情** | 恋爱反复遇渣男/渣女、职场被甩锅、原生家庭纠缠、抑郁焦虑 |
| **先天配角角色** | 父母 = 「情感忽视者」或 「条件式关爱者」 |
| **后天配角角色** | 伴侣 = 「伤害者/索取者」；朋友 = 「利用者」；上司 = 「压榨者」 |

---

### 3. 隐士

| 项目 | 内容 |
|-----|------|
| **维度组合** | 安全型 + 疏离型 + 自卑型 + 停滞型 |
| **人设描述** | 「我必须稳定，世界与我无关，我不够好，我已定型」 |
| **核心信念** | 改变是危险的，独处最安全 |
| **编剧写的剧本** | 固守舒适区、拒绝新机会、情感隔离、自我设限 |
| **常见剧情** | 多年不升职、回避亲密关系、沉迷虚拟世界、错失人生机遇 |
| **先天配角角色** | 父母 = 「过度保护者」或 「情感疏离者」 |
| **后天配角角色** | 伴侣 = 「难以靠近的室友」；朋友 = 「偶尔联系的网友」；同事 = 「互不干扰的平行线」 |

---

### 4. 控制狂

| 项目 | 内容 |
|-----|------|
| **维度组合** | 成就型 + 掌控型 + 完美型 + 追赶型 |
| **人设描述** | 「我必须成功，世界可控制，我不能有缺点，必须赶上」 |
| **核心信念** | 失控 = 失败，必须掌控一切 |
| **编剧写的剧本** | micromanagement、焦虑失眠、计划被打乱就崩溃、难以信任他人 |
| **常见剧情** | 团队离心、伴侣窒息、孩子叛逆、健康亮红灯 |
| **先天配角角色** | 父母 = 「高控制者」（习得）或 「无力者」（反向形成） |
| **后天配角角色** | 伴侣 = 「被控制的傀儡」→「反抗的叛逆者」；下属 = 「阳奉阴违的执行者」；孩子 = 「压抑的继承者」或 「彻底的叛逆者」 |

---

### 5. 受害者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 关系型 + 受害者型 + 自卑型 + 宿命型 |
| **人设描述** | 「我必须被爱，世界伤害我，我不够好，一切注定」 |
| **核心信念** | 我是无辜的，痛苦不是我的错 |
| **编剧写的剧本** | 重复创伤、吸引加害者、无力改变、寻求同情 |
| **常见剧情** | 职场被霸凌、感情被欺骗、经济被控制、长期抑郁 |
| **先天配角角色** | 父母 = 「加害者」或 「拯救者-加害者循环」 |
| **后天配角角色** | 伴侣 = 「施虐者」或 「拯救者-疲惫离开者」；朋友 = 「同情但无力者」；社会 = 「冷漠的旁观者」 |

---

### 6. 表演者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 独特型 + 自恋型 + 完美型 + 探索型 |
| **人设描述** | 「我必须特别，我优于他人，我不能有缺点，人生是体验」 |
| **核心信念** | 被看见 = 存在，平凡 = 死亡 |
| **编剧写的剧本** | 不断追求新鲜感、关系 superficial、内心空虚、形象管理疲惫 |
| **常见剧情** | 频繁换工作/城市/伴侣、社交媒体成瘾、中年危机、身份崩塌 |
| **先天配角角色** | 父母 = 「conditional love 给予者」或 「忽视真实需求的赞美者」 |
| **后天配角角色** | 伴侣 = 「观众/粉丝」→「厌倦的离开者」；朋友 = 「仰望者」或 「竞争者」；公众 = 「追捧或唾弃的群众」 |

---

### 7. 拯救者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 服务型 + 合作型 + 自卑型 + 创造型 |
| **人设描述** | 「我必须有用，世界是伙伴，我不够好（除非被需要），未来可创造」 |
| **核心信念** | 被需要 = 有价值，帮助他人 = 帮助自己 |
| **编剧写的剧本** | 过度付出、忽视自己、吸引索取者、 burnout |
| **常见剧情** | 职业倦怠、关系不平等、身体垮掉、怨恨但无法停止 |
| **先天配角角色** | 父母 = 「需要被照顾的弱者」或 「牺牲型的榜样」 |
| **后天配角角色** | 伴侣 = 「被拯救的依赖者」→「怨恨的独立者」；朋友 = 「不断求助者」；同事 = 「甩锅者」 |

---

### 8. 漫游者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 独特型 + 疏离型 + 迷失型 + 探索型 |
| **人设描述** | 「我必须特别，世界与我无关，我不知道我是谁，人生是旅程」 |
| **核心信念** | 答案在路上，停留 = 窒息 |
| **编剧写的剧本** | 不断更换方向、难以深入、关系短暂、存在主义空虚 |
| **常见剧情** | 多年漂泊无定、职业履历零散、亲密关系恐惧、晚年回望迷茫 |
| **先天配角角色** | 父母 = 「缺席者」或 「无法理解的旁观者」 |
| **后天配角角色** | 伴侣 = 「短暂的同行者」；朋友 = 「散落各地的旧识」；社会 = 「无法归类的边缘人」 |

---

### 9. 战士

| 项目 | 内容 |
|-----|------|
| **维度组合** | 成就型 + 战斗型 + 真实型 + 创造型 |
| **人设描述** | 「我必须成功，世界是战场，我接纳自己，未来由我创造」 |
| **核心信念** | 战斗是本能，但我不被战斗定义 |
| **编剧写的剧本** | 积极竞争、直面冲突、从失败学习、建立同盟 |
| **常见剧情** | 创业成功或光荣失败、深厚战友情、健康竞争关系、持续成长 |
| **先天配角角色** | 父母 = 「坚强的后盾」或 「共同战斗的战友」 |
| **后天配角角色** | 伴侣 = 「并肩作战的搭档」；朋友 = 「可靠的盟友」；对手 = 「尊重的敌人」 |

---

### 10. 治愈者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 关系型 + 合作型 + 真实型 + 创造型 |
| **人设描述** | 「我需要爱，世界是伙伴，我接纳自己，未来可创造」 |
| **核心信念** | 爱是流动的，给予和接受同样重要 |
| **编剧写的剧本** | 健康亲密关系、互相支持的朋友圈、有意义的工作、内心平静 |
| **常见剧情** | 稳定的伴侣关系、深厚的友谊、职业满足感、帮助他人成长 |
| **先天配角角色** | 父母 = 「真实的关爱者」（无论是否完美） |
| **后天配角角色** | 伴侣 = 「平等的同行者」；朋友 = 「互相滋养的知己」；社区 = 「贡献者和受益者」 |

---

### 11. 观察者

| 项目 | 内容 |
|-----|------|
| **维度组合** | 安全型 + 疏离型 + 真实型 + 探索型 |
| **人设描述** | 「我需要稳定，世界与我无关，我接纳自己，人生是旅程」 |
| **核心信念** | 观察即参与，深度优于广度 |
| **编剧写的剧本** | 深耕专业领域、少而精的关系、独立思考、从容节奏 |
| **常见剧情** | 领域专家、长期稳定的少数友谊、晚年智慧、影响深远但低调 |
| **先天配角角色** | 父母 = 「尊重边界的支持者」 |
| **后天配角角色** | 伴侣 = 「独立但亲密的伴侣」；朋友 = 「深度对话的知己」；学生/读者 = 「受益于其智慧的人」 |

---

### 12. 觉醒者

| 项目 | 内容 |
|-----|------|
| **维度组合** | （可变的组合，核心是「真实型」主导） |
| **人设描述** | 「我在觉察我的人设，我可以改写剧本」 |
| **核心信念** | 我是编剧，不是剧本 |
| **编剧写的剧本** | 主动选择困难、设计成长挑战、筛选滋养性关系、持续自我更新 |
| **常见剧情** | 从任何原型转型而来、有意识地设计人生、帮助他人觉醒、活出多种可能性 |
| **先天配角角色** | 父母 = 「被重新理解的真实的人」（角色已改写） |
| **后天配角角色** | 伴侣 = 「共同成长的伙伴」；朋友 = 「互相启发的觉醒者」；世界 = 「共同创作的舞台」 |

## 改写路径

### 从痛苦原型到觉醒者的路径

| 当前原型 | 核心改写 | 新原型 |
|---------|---------|--------|
| 孤勇者 | 从「必须成功」→「可以成功也可以失败」 | 战士 |
| 讨好者 | 从「必须被爱」→「我值得被爱，无需证明」 | 治愈者 |
| 隐士 | 从「改变是危险的」→「改变是成长」 | 观察者 |
| 控制狂 | 从「必须控制」→「可以影响，接纳不确定」 | 战士 |
| 受害者 | 从「我是无辜的」→「我有选择的权力」 | 觉醒者 |
| 表演者 | 从「被看见=存在」→「我存在，无需证明」 | 治愈者 |
| 拯救者 | 从「被需要=有价值」→「我有价值，无需拯救他人证明」 | 治愈者 |
| 漫游者 | 从「答案在路上」→「答案也在当下」 | 观察者或觉醒者 |

## 测试设计参考

**12题，每题对应一种原型**
- 用户选择最共鸣的描述
- 或设计场景，看反应倾向

**结果呈现**
- 主要原型（当前主导人设）
- 次要原型（潜在或过去人设）
- 觉醒路径（如何从当前原型向觉醒者发展）
