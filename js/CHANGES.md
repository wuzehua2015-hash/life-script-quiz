## 一、角色原型分类修改详情

### 1. 甄嬛 (《甄嬛传》)
**修改前**: 孤勇者 + 控制狂（重复出现）
**修改后**: 控制狂（仅保留一处）

**修改理由**:
- 甄嬛的核心特征是"精于算计、掌控后宫、步步为营"
- 她的驱动力是relationship（渴望被爱和认可），而非achievement
- 孤勇者原型强调独自战斗，而甄嬛是善于利用关系和控制局面的角色

**Cast描述修正**:
```javascript
// 修改前（错误）
{ name: "高期待的施压者", desc: "他们的认可永远差一点点" }

// 修改后（正确）
{ name: "温厚但无力的父亲", desc: "甄远道疼爱你但无法在宫廷保护你" },
{ name: "教礼仪但非施压的母亲", desc: "教你生存之道，而非逼迫你争宠" }
```

### 2. 林黛玉 (《红楼梦》)
**修改前**: 讨好者
**修改后**: 隐士

**修改理由**:
- 林黛玉的核心特征是"敏感孤傲、远离世俗、才华横溢"
- 她是寄人篱下的孤女，但并非讨好型人格
- 她的悲剧是孤傲和敏感，而非讨好
- 更符合隐士的`drive: unique, world: detachment, self: narcissism`

**Cast描述修正**:
```javascript
// 修改前（错误）
{ name: "情感忽视者", desc: "你的感受很少被看见" }

// 修改后（正确）
{ name: "早逝的才女母亲贾敏", desc: "你继承了她的才华和敏感" },
{ name: "无法保护你的父亲林如海", desc: "他把你送到外祖母家后离世" }
```

### 3. 祥子 (《骆驼祥子》)
**修改前**: 讨好者
**修改后**: 受害者

**修改理由**:
- 祥子是老舍笔下经典的受害者形象
- 他被命运打击、失去希望、最终堕落
- 符合受害者的`drive: relationship, world: victim, self: inferiority, time: fate`

### 4. 魏璎珞 (《延禧攻略》)
**修改前**: 表演者
**修改后**: 控制狂

**修改理由**:
- 魏璎珞的核心特征是"精于算计、掌控局面、步步为营"
- 她善于控制局势，利用人心
- 不是表演者（不在意他人关注），而是控制者

### 5. Tony Stark (《钢铁侠》)
**修改前**: 孤勇者
**修改后**: 表演者

**修改理由**:
- Tony Stark的核心特征是"需要关注、展示才华、享受聚光灯"
- 他说"I am Iron Man"是典型的表演者宣言
- 符合表演者的`drive: unique, world: narcissism, self: narcissism`

### 6. Arya Stark (《权力的游戏》)
**修改前**: 漫游者
**修改后**: 战士

**修改理由**:
- Arya的核心特征是"复仇、战斗、不屈"
- 她学习暗杀，完成复仇清单
- 不是漫游者（探索世界），而是战士（为复仇而战）

### 7. 王漫妮 (《三十而已》)
**修改前**: 漫游者
**修改后**: 表演者

**修改理由**:
- 王漫妮是奢侈品柜姐，追求更好生活
- 她在意形象和社会地位
- 符合表演者的特征，而非漫游者

### 8. 程蝶衣 (《霸王别姬》)
**修改前**: 孤勇者
**修改后**: 孤勇者（保留，但需说明）

**修改理由**:
- 程蝶衣是"为艺术献身、不疯魔不成活"
- 他既是孤勇者（独自执着），也是表演者（戏痴）
- 经讨论保留在孤勇者，因为他的核心是"极致追求"而非"表演给他人看"

## 二、新增角色

### 1. 闰土 (《故乡》) - 受害者
**理由**: 鲁迅笔下经典的被生活压垮的农民形象，补充受害者原型的中国文学代表

### 2. 孙悟空 (《西游记》) - 漫游者
**理由**: 从石猴到斗战胜佛的漫游旅程，是中国文学中最经典的漫游者形象

### 3. 鲁智深 (《水浒传》) - 漫游者
**理由**: 从提辖到和尚的江湖漫游，最终顿悟，补充漫游者的中国武侠代表

### 4. 吴用 (《水浒传》) - 观察者
**理由**: 智多星，善于观察局势、谋划策略，是观察者的经典代表

## 四、Cast描述统一规范

### 先天配角（父母）描述规范

根据角色实际家庭背景，分为四类：

#### 1. 高期待施压型（孤勇者、控制狂）
- 高期待的施压者
- 不理解但无奈的支持者

#### 2. 温厚支持型（治愈者、战士）
- 真实的关爱者
- 情感支持的榜样

#### 3. 缺席/无力型（受害者、隐士）
- 缺席者/早逝者
- 无法保护你的无力者

#### 4. 寄人篱下型（林黛玉）
- 早逝的母亲
- 无法保护你的父亲

### 后天配角描述规范

每个原型有统一的后天配角描述：

| 原型 | 后天配角1 | 后天配角2 | 后天配角3 |
|------|-----------|-----------|-----------|
| 孤勇者 | 被忽视的同行者 | 仰望但无法亲近的追随者 | 必须击败的敌人 |
| 讨好者 | 伤害者/索取者 | 利用者 | 拯救者-疲惫离开者 |
| 隐士 | 难以靠近的室友 | 偶尔联系的网友 | 互不干扰的平行线 |
